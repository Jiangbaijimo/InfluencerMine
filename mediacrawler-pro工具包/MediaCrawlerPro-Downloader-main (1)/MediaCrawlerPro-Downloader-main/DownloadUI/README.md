# MediaCrawlerProDownloader 自媒体平台下载器UI

一个基于 Electron + Vue3 + TypeScript 开发的自媒体平台内容下载器UI。支持抖音、小红书等平台的视频和图片下载。


## 技术栈

- Electron
- Vue
- TypeScript
- Vite
- Element Plus 


## 安装

### 安装依赖
> 依赖nodejs，nodejs的版本大于等于16
npm install

### 启动开发服务器
npm run dev
